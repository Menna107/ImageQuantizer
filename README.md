# ImageQuantizer

**ImageQuantizer** is a Python desktop application for compressing and decompressing images using **Vector Quantization** with **K-Means clustering**. The app provides an interactive **Tkinter GUI** with support for dark mode, allowing users to preview images before and after compression.

---

## Features

- Compress images using vector quantization
- Decompress compressed `.npz` image files
- Set custom compression parameters:
  - Block height & width
  - Number of bits
  - Output file name
- Preview original, compressed, and decompressed images
- Dark-themed modern GUI
- Multi-threaded operations for smooth UI
